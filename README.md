# AI Fitness Planner 🏋️

Personalized Indian diet and workout plan powered by AI.

## Features
- BMI Calculator
- AI Generated Diet Plan
- AI Generated Workout Plan  
- Chat with AI Coach
- Indian food suggestions only

## Built With
- Python
- Streamlit
- Groq AI (llama-3.3-70b)

## How to Run
1. Clone this repository
2. Install requirements: pip install streamlit groq
3. Add your Groq API key in main.py
4. Run: streamlit run main.py
